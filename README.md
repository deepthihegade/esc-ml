# Spam Classifier Streamlit App

This project is a machine learning-powered Streamlit web application that classifies emails and SMS messages as spam or not spam. It uses TF-IDF vectorization along with an ensemble of classifiers for accurate spam detection.

## Features
- Real-time spam classification via a simple web interface
- Text preprocessing with NLTK and TF-IDF
- Ensemble model for improved accuracy
- Easy to run locally and deploy
